产品简介

甲醛检测仪

甲醛检测仪是一款高精度甲醛浓度检测仪器。采用英国原装进口传感器，配合卓越工艺，给你最极致的体验。

甲醛检测仪（移动版）

甲醛检测仪（移动版）可以通过收集数据口进行供电及数据传输。检测数据通过手机app显示。

甲醛检测仪（WIFI加强版）

甲醛检测仪（WIFI加强版）在移动版基础上集成WIFI模块，可以远程控制设备及获取数据。没有网络也可以工作~

PM2.5检测仪

PM2.5检测仪是一款高精度PM2.5检测仪器。采用原装进口传感器，配合卓越工艺，给你最极致的体验。

PM2.5检测仪（移动版）

PM2.5检测仪（移动版）基于基础版研发，可以通过手机数据库进行供电及数据传输。监测数据通过app查看。

PM2.5检测仪（WIFI加强版）

PM2.5检测仪（WIFI加强版）在移动版基础上集成WIFI模块，可以远程控制设备及获取数据。没有网络也可以工作~
